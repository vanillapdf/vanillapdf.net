# Changelog

Release notes for Vanilla.PDF .NET are published on the
[GitHub Releases page](https://github.com/vanillapdf/vanillapdf.net/releases),
curated per tag as part of the release process. That page is the authoritative
changelog — this file exists only so tooling and contributors that expect a
`CHANGELOG.md` have somewhere to look.

- **Latest release**: https://github.com/vanillapdf/vanillapdf.net/releases/latest
- **All releases**: https://github.com/vanillapdf/vanillapdf.net/releases
- **Compare any two versions**: `https://github.com/vanillapdf/vanillapdf.net/compare/vX.Y.Z...vA.B.C`

Each release wraps a specific version of the native Vanilla.PDF engine. Engine-level
changes are tracked separately on the native project's
[releases page](https://github.com/vanillapdf/vanillapdf/releases).
